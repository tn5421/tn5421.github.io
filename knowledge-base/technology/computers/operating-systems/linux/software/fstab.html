fstab

FStab, or the file system table, is a file on linux, typically located at `/etc/fstab`.

This file tells linux where, why, and how to mount various disks.

Example of a working fstab file

# <file system>  <mount point> <type>   <options>  <dump> <pass>
UUID=1175-49F5    /boot/efi     vfat    umask=0077  0 2
UUID=5cc8fd0b-8b93-495c-bc51-70ca2bc05237 /  ext4 defaults,noatime 0 1
UUID=1597ac1b-797c-48ab-b248-da290784da8b swap  swap  defaults,noatime 0 0
tmpfs     /tmp   tmpfs   defaults,noatime,mode=1777 0 0

Known options and definitions

umask, or usermask, defines user permissions. uses a different scheme than the usual octet/octal, as 0022 is usually required for regular user access.

defaults is a set of relatively sensible defaults for mounting disks, including but not limited to read and write permissions.
