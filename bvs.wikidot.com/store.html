Store

The store is a place to buy and sell items for Ryo. It is available from the "Go Shopping" button on the main menu, and also available at the start of missions. It appears to be run by MC Stripeypants.

Description from BvS

Let's see, you've got to get yourself some HOLY CRAP THOSE ARE SOME STRIPEY PANTS! "What'dya need, fella? I don't got all day.."

Buy the tools you need to perform powerful Jutsu, and sell items you find on Missions!
