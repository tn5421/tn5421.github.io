BvS-Uzuki 23 Feb 2008 02:00

Hey there!

We've been working together on this Wiki for a bit now, and for the most part, the only person you've really had a chance to talk with is me. So I figured setting up this forum would help to develop a sense of community in the Wiki. And what better way to get to know each other than to introduce each other! So, I'll go ahead and start.

So I'm BvS-Uzuki, but you guys can call me Uzuki. I've been playing BvS for a few weeks now and thought it was a great game, but that it was really hard to find information about it. So I went ahead and setup this Wiki to help out. My current character is Uzuki from Murder Village. I'll have to say, I've learned a ton about the game since I set things up, so I'm very much glad I did.

I'd include my BvS signature image, but I can't figure out quite how to do that yet ;)
