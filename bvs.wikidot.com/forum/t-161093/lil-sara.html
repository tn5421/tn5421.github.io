How the signature works:

You use a include in which you only insert your "Character ID" (it is the last 5 digits of the Billy/image link. example:"00000") where it says "your ID here".

how you did it:

[[=]]
[[include include:Signature image http://www.animecubed.com/billy/?73962 link="http://www.animecubed.com/billy/?73962"]]
|id=YourIdHere

[[/=]]

how it should be done:

[[=]]
[[include include:Signature
|id=73962
]]
[[/=]]

this discussion page wont allow me to show the result, but it should work as I tested it on another page.

I also noticed that you didn't know the tab code so here it is:

[[tabview]]
[[tab TABNAME 1]]
Something
[[/tab]]
[[tab TABNAME 2]]
Somthing else
[[/tab]]
[[/tabview]]

result:

  • TABNAME 1
  • TABNAME 2

Something

Something else

Emtpy Tab

An empty tab is created by using a tab without any text:

[[tabview]]
[[tab Empty Tab]]
[[/tab]]
[[tab Not Empty Tab]]
Not Empty
[[/tab]]
[[/tabview]]
  • Empty Tab
  • Not Empty Tab


Not Empty

Overlap

The easiest way to prevent the ally tab from overlapping the avatar is to fill in text between the top and the ally tab or use " _" to make empty rows (if used on the start of a row)
